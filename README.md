## 安装与运行

``` bash
# 安装依赖
npm install 
# 客户端运行访问localhost:4200
控制台进入项目根目录下输入：npm run start
# 服务端运行，控制台进入server/bin目录下输入：
node www

# build for production and view the bundle analyzer report
npm run build --report
